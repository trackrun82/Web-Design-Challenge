Summary: Latitude vs X

The purpose of this project was to analyze if weather changed in proximity to the equator. To accomplish this analysis, data was pulled from the OpenWeatherMap API to assemble a dataset of over 500 cities.

After gathering the dataset, Matplotlib was used to create various plots comparing weather elements and latitude. Factors looked into included: temperature, cloudiness, wind speed, and humidity. Further analysis was also done to see whether a city was in the Northern or Southern hemisphere played a factor. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
